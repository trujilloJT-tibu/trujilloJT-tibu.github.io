Ecosistema Core Relacional (Estructura SIEM Local)

A diferencia de scripts aislados, este entorno funciona de forma integrada compartiendo telemetría en una base de datos centralizada. Esto permite correlacionar eventos técnicos y almacenar alertas en tiempo real.

Esquema del Diccionario SQL (ciberdefensa.db):
  • incidentes: Mapea IP, timestamp, payload e indicadores OWASP capturados por el parser.
  • alertas_red: Registra métricas de saturación y eventos de denegación de servicio (DoS) en entornos IoT.
